== Description ==

基于加密稳定币 FOUSDT 的 WordPress 付费阅读插件，它允许你将文章部分或者全部内容设置为付费后才可阅读插件。特点如下：

- 采用绑定美元的稳定加密货币 USDT 在 FIBOS 链上的版本作为支付接口。这样可以在全球范围内快速收款。
- 通过发布文章是的扩展字段进行费用设置，新旧编辑器都可以支持。
- 本插件会调用 api.fowallet.net 查询 FIBOS 区块链确认到账情况；使用 qr.topscan.com 显示支付二维码。

== Frequently Asked Questions ==

= 什么是 FIBOS =

FIBOS 是一个可以去中心化跨链、可流动的 ETH、USDT、EOS 资产和稳定币的区块链。访问 fibos.io 可以了解更多。

= 如何进行支付 =

需要下载 FIBOS 钱包，网址为 https://wallet.fo/
